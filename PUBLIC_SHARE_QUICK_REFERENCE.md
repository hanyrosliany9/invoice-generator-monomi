## ğŸ› Troubleshooting

### Image Won't Load
- **Solution**: Image has automatic retry (3 attempts)
- If still fails: Shows "Image Not Found" placeholder
- Try refreshing browser

### Video Won't Play
- **Solution**: Check browser compatibility (Chrome/Edge/Firefox/Safari)
- Ensure codec is supported (H.264 widely supported)
- Try downloading file

### Keyboard Shortcuts Not Working
- **Solution**: Make sure you're not typing in search box
- Click outside input fields first
- Close any open modals except lightbox

### Comparison Button Disabled
- **Cause**: Wrong number of assets selected
- **Solution**: Select 2-4 assets (not 1, not 5+)

### Can't Select Assets
- **Solution**: Click "Select Items" button in top-right
- OR: Just start clicking checkboxes (auto-enters mode)

### Download Failed
- **Solution**: Check browser download settings
- Allow popups from this site
- Try downloading smaller batches
